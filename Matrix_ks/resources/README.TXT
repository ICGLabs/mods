Section A.    O V E R V I E W

Note:  This is a basic screen saver which displays near-authentic
       Matrix font characters in the same animation sequece
       used in the movie.  There are various color options too!
            It also optionally displays your system
          statistics including system model, cpu, Memory Usage,
	  CPU Usage, and more!
          :)

          It also supports multiple monitors!
          It also allows you to set password protection via the
          screen saver SETTINGS dialog in your windows Control
          Panel display options.

          I welcome your feedback.

This screen saver is donationware (for NON-commerical use).
Feel free to send me a donation if you plan to keep it.
Donations motivate me to make more of these and improve them.
:)


REQUIREMENTS:
Disk space required after install is approx.  1.8 meg.


------------------------------------------------------------
Section B.  HOW TO INSTALL:

1)..Run the install.exe  (double-click it). This is self-
          extracting installation program.

2)..That's it!
          You're ready to set some options!
          1)..Go to your Control Panel, select Display.
          2)..Select the Screen Savers tab and select the
                    "Matrix_ks".  Then click the Settings box.

Enjoy!




------------------------------------------------------
Section C. How to UNinstall it:

          1)..Go to your Control Panel, click Add/Remove Programs
		and look for Matrix-ks.
